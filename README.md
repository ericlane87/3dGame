# Hood Chronicles 3D

Browser-based 3D shooter prototype (single-player) built with Three.js.

## How to run

From `projects/hood-chronicles-3d`:

```bash
python3 -m http.server 5173
```

Then open `http://localhost:5173` in your browser.

## Controls

Desktop:
- Click to lock mouse
- Move: WASD
- Sprint: Shift
- Shoot: Left click
- Map: M (only from garage after returning)

Mobile:
- Move: left joystick
- Look: drag right side
- Shoot: Shoot button
- Map: Map button (only from garage after returning)

## Current scope

- Garage safe house, neighborhood, and outskirts
- Basic handgun shooting
- Zombies with vision cone + sound radius (2D sprite placeholder with front/back/side)
- Pickups: food and med
- Resource storage at base
- Map travel to unlocked location
- Scientist objective trigger (placeholder)
